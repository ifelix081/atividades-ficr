# 📚 Atividades FICR

Este repositório foi criado com o objetivo de centralizar e organizar todas as atividades desenvolvidas ao longo do curso na FICR.

## 🌐 Acesse o site

As atividades também estão disponíveis de forma visual através do GitHub Pages:

👉 https://ifelix081.github.io/atividades-ficr/

## 📁 Estrutura do repositório

As atividades estão organizadas em pastas separadas, onde cada uma representa um exercício, projeto ou prática específica.

## 🎯 Objetivo

Este repositório tem como finalidade:

* Armazenar todas as atividades realizadas
* Facilitar a visualização e acesso aos projetos
* Servir como portfólio de aprendizado e evolução

## 🚀 Atualizações

Novas atividades serão adicionadas continuamente ao longo do curso.
